# CrunchyWrap
CrunchyWrap is an Android app that allows you to stream content from CrunchyRoll. It makes use of the private CrunchyRoll's API and is only a PoC of how to interact with REST APIs.
[Source Code](https://github.com/joe2k01/CrunchyWrap)
[Documentation](http://crunchywrap.altervista.org/)

# DeskWidget
DeskWidget is a "smart" desk clock. It displays the time and what's the weather like in any user defined location. It uses an Atmega328P-PU to drive an E-INK display and retrievs data from the internet using an ESP-01.
It is not the most effective way to approach this problem, as an ESP32 would have been an all in one solution, but those were the parts that I had lying around.
[Source Code](https://github.com/joe2k01/DeskWidget)
